# Strategy overview

## North-star principle

**Use LLMs to remove friction â€” not to replace ownership.**

A senior AI documentation strategy focuses on:
- **outcomes** (time-to-first-success, support deflection, adoption)
- **constraints** (accuracy, privacy, compliance, maintainability)
- **operational reality** (review capacity, release cadence, toolchain)

## Maturity levels (example)

0. **Ad hoc**: individuals use chat tools, no shared standards  
1. **Assisted**: shared prompt patterns + human review  
2. **Instrumented**: evaluation datasets + metrics + CI gates  
3. **Integrated**: RAG over trusted sources + feedback loops  
4. **Optimized**: continuous improvement + cost/quality optimization
